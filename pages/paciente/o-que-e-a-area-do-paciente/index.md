**Dados do paciente** - Aba com os dados do paciente. Podendo alterar informações.

</p>

**Anamnese** - Podendo visualizar, editar e imprimir a anamnese.

  <img alt="Área do paciente - Aba dados pessoais paciente" src="o-que-e-a-area-do-paciente-img-03.png" width="800">

**Plano de Tratamento** - Lista todos os planos de tratamento do paciente.

  <img alt="Área do paciente - Aba dados pessoais paciente" src="o-que-e-a-area-do-paciente-img-04.png" width="800">
</p>

**Atestado** - Lista com todos atestados já feitos, podendo adicionar um novo e imprimir.

  <img alt="Área do paciente - Aba dados pessoais paciente" src="o-que-e-a-area-do-paciente-img-05.png" width="800">

**Receitas** - Lista de todas as receitas, podendo adicionar um novo, remover e imprimir.

  <img alt="Área do paciente - Aba dados pessoais paciente" src="o-que-e-a-area-do-paciente-img-07.png" width="800">
